예제53-1 터치센서의 동작을 시리얼플로터로 관찰하자
 * 터치센서 2개는 각각 아래와 같이 연결했다.
 * 1번 터치센서 : D2
 * 2번 터치센서 : D3
 * (결과 눌렀을때 HIGH)
예제53-2 터치센서로 LED 1개 제어해보자!
 * 터치센서 2개는 각각 아래와 같이 연결했다.
 * 1번 터치센서 : D2
 * 2번 터치센서 : D3
 * LED는 D4연결하고 1번센서를 터치하면 켜지고
 * 2번센서를 터치하면 꺼지게 하라!
예제53-3 터치센서로 FND디스플레이 업/다운카운트 해보자
 * 터치센서 2개는 각각 아래와 같이 연결했다.
 * 1번 터치센서 : D2
 * 2번 터치센서 : D3
 * FND디스플레이(TM1637) 아래와 같이 연결했다
 * CLK : D4
 * DIO : D5
예제53-4 터치센서로 솔레노이드를 작동시켜보자
 * 터치센서 2개는 각각 아래와 같이 연결했다.
 * 1번 터치센서 : D2
 * 2번 터치센서 : D3
 * 솔레노이드는 직접 제어가 힘들어서 모스펫으로 해보자
 * 모스펫의 시그널핀을 D4에 연결했다
예제53-5 첫번쨰 터치센서를 누르면 LED가 점점 밝아지고
 * 두번쨰 터치센서를 누르면 LED가 점점 어두워지게하라!
 * 터치센서 2개는 각각 아래와 같이 연결했다.
 * 1번 터치센서 : D2
 * 2번 터치센서 : D3
 * LED는 D5에 연결했다.(PWM포트이기 때문)
